# Domain: Finance Research

> **Status: Coming soon.** This domain is a placeholder.

Planned skills for quantitative research, factor analysis, portfolio construction, and financial modeling workflows.

## Proposed Skills

| Command | Purpose |
|---------|---------|
| `/finance-analyst` | Persona — quantitative researcher mindset, risk-first thinking |
| `/finance-data-audit` | Data quality checks for financial time series (gaps, splits, survivorship bias) |
| `/finance-factor-research` | Alpha factor construction, IC analysis, factor decay |
| `/finance-backtest` | Walk-forward backtesting, avoiding look-ahead bias |
| `/finance-portfolio` | Mean-variance optimization, risk parity, position sizing |
| `/finance-risk` | VaR, CVaR, drawdown analysis, stress testing |
| `/finance-report` | Tear sheet generation, performance attribution |

## Want to contribute this domain?

Open a PR with a `skills/finance-<name>/SKILL.md` file. See the Kaggle domain for skill format examples.
